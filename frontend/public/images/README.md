# Logo Images

This directory contains logo images for the SoftArt AI HUB application.

## Required Logo Files:

### 1. Main Logo (`logo.png`)
- **Location**: `/images/logo.png`
- **Usage**: Header navigation (48x48px recommended)
- **Format**: PNG with transparent background
- **Max Size**: 100KB
- **Dimensions**: Square aspect ratio, minimum 64x64px

### 2. Small Logo (`logo-small.png`)
- **Location**: `/images/logo-small.png`
- **Usage**: Footer branding (32x32px recommended)
- **Format**: PNG with transparent background
- **Max Size**: 50KB
- **Dimensions**: Square aspect ratio, minimum 32x32px

## Logo Guidelines:

- **File Format**: PNG (preferred) or SVG
- **Background**: Transparent preferred, or match your site's background
- **Colors**: Should work well on dark backgrounds (#111827)
- **Scalability**: Logo should be clear at small sizes
- **Brand Consistency**: Use official SoftArt AI HUB branding

## Fallback Behavior:

If logo images are not found, the application will automatically fall back to displaying "AI" text instead.

## Adding Your Logo:

1. Place your logo files in this directory
2. Name them exactly as specified above
3. Ensure they meet the size and format requirements
4. Refresh your browser to see the changes

## Optimization Tips:

- Use tools like TinyPNG or ImageOptim to compress images
- Test logos at different sizes to ensure readability
- Consider creating both light and dark versions if needed
- Use vector formats (SVG) when possible for crisp scaling
